The Leaderboard API returns a global ranking of UpForge's most active learners. Even if you're outside the top 100, your personal stats and rank are always included in the response so you can see exactly where you stand.

---

## GET /api/leaderboard

Returns the global leaderboard (up to 100 users ordered by total XP, highest first) plus a snapshot of your own stats and rank.

### Response fields

<ResponseField name="entries" type="LeaderboardEntry[]" required>
  Ordered array of the top users (maximum 100). Sorted descending by `points`.

  <Expandable title="LeaderboardEntry object fields">
    <ResponseField name="id" type="string">UpForge internal user UUID.</ResponseField>
    <ResponseField name="name" type="string | null">The user's display name.</ResponseField>
    <ResponseField name="username" type="string | null">The user's unique username.</ResponseField>
    <ResponseField name="avatarUrl" type="string | null">URL of the user's profile picture, or `null` if none is set.</ResponseField>
    <ResponseField name="points" type="number">Total XP the user has accumulated.</ResponseField>
    <ResponseField name="level" type="number">The user's current level, derived from their total XP.</ResponseField>
    <ResponseField name="currentStreak" type="number">Number of consecutive days the user has completed at least one lesson.</ResponseField>
    <ResponseField name="rank" type="number">1-based position in this response (rank 1 = highest XP).</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="userRank" type="number | null" required>
  Your 1-based rank within the top-100 response. `null` if you are outside the top 100 or if your account could not be found.
</ResponseField>

<ResponseField name="userPoints" type="number" required>
  Your current total XP. Returns `0` if your account is not found.
</ResponseField>

<ResponseField name="userLevel" type="number" required>
  Your current level. Returns `1` if your account is not found.
</ResponseField>

<ResponseField name="userStreak" type="number" required>
  Your current daily streak. Returns `0` if your account is not found.
</ResponseField>

<ResponseField name="totalUsers" type="number" required>
  Number of users actually included in the `entries` array (≤ 100).
</ResponseField>

### Error responses

| Status | Meaning |
|--------|---------|
| `401`  | You are not authenticated. |

<Note>
  If a server error occurs, the endpoint returns a safe fallback payload with empty `entries` and zeroed-out user stats rather than a `5xx` error code. Always check whether `entries` is non-empty before rendering a leaderboard UI.
</Note>

</Tip>
